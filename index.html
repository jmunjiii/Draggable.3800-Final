draggable JS

a lightweight, responsive, modern drag & drop library

Draggable is a flexible drag-and-drop library that lets you start with the basics and add features as needed. At its core, it provides drag-and-drop functionality, quick DOM reordering, accessible HTML structure, and a set of events to work with.
